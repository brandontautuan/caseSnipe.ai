# Agent GIFs

Place transparent-background GIFs here for the Prosecutor and Defense avatars. Each GIF is shown based on the agent's current state.

## Required files

### Prosecutor
| File | When shown |
|------|------------|
| `prosecutor-idle.gif` | Agent is idle (waiting for trial) or resting their case |
| `prosecutor-speaking.gif` | Agent is arguing / delivering speech |
| `prosecutor-objecting.gif` | Agent is filing a motion or cross-examining |
| `prosecutor-researching.gif` | Agent is searching, looking up precedents, or requesting evidence |

### Defense
| File | When shown |
|------|------------|
| `defense-idle.gif` | Agent is idle (waiting for trial) or resting their case |
| `defense-speaking.gif` | Agent is arguing / delivering speech |
| `defense-objecting.gif` | Agent is filing a motion or cross-examining |
| `defense-researching.gif` | Agent is searching, looking up precedents, or requesting evidence |

## Tips
- Use GIFs with transparent backgrounds for best effect
- Recommended size: ~200–400px width for good quality at 2x
- If a GIF is missing, the avatar will hide the broken image
